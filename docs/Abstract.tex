\section*{ABSTRACT}
\addcontentsline{toc}{section}{\numberline{}ABSTRACT}

This project presents FootField, a comprehensive B2B2C Software as a Service (SaaS) platform designed to digitize the management and booking operations of football facilities. The system bridges the gap between facility owners (Tenants), platform administrators (Providers), and end-users (Customers) within a unified ecosystem. Built on a robust Node.js backend with a multi-tenant MySQL architecture, FootField leverages Google Gemini 2.5 Flash for conversational AI booking via function calling, and utilizes Firebase Cloud Messaging (FCM) for real-time notifications. The frontend follows a modular design system for high performance, while the mobile applications, powered by Ionic Capacitor, integrate native hardware features such as QR code scanning and wireless printing. This hybrid approach ensures high scalability, visual consistency, and a seamless cross-platform user experience.

\textbf{Keywords:} SaaS, Multi-tenancy, Conversational AI, Function Calling, Node.js, Capacitor, FCM.
